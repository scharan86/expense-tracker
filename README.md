# Expense Tracker

A simple expense tracking website I built for my college Web Technologies course. It lets you add expenses, categorize them, and see where your money goes.

![WhatsApp Image 2025-11-16 at 11 41 00 AM](https://github.com/user-attachments/assets/7255c974-c1d0-4364-a235-79b92db8ee39)

## What it does

- Add and delete expenses
- View all your spending in a table
- See a dashboard with stats
- Generate reports by category
- Filter and sort your expenses

## Built with

- HTML, CSS, JavaScript for the frontend
- PHP for the backend
- MySQL for the database
- Tokyo Night color scheme (because it looks cool)
- JetBrains Mono font

## Setup

Install Apache, PHP, and MySQL on your system. Import the database from `database/setup.sql`, copy the project to your web server directory, and open it in your browser at `localhost/expense-tracker`.

## Note

This is an educational project for learning web development basics. It covers HTML forms, CSS layouts, JavaScript DOM manipulation, PHP database operations, and basic CRUD functionality.
